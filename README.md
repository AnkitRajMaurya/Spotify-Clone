# Spotify Clone

This is a simple Spotify-style web UI created for educational purposes only. It is a demo project and is not affiliated with or endorsed by Spotify.

## Disclaimer

This project is for educational purposes only. It does not use the official Spotify API or brand assets beyond a UI-inspired design.

## Safety note

This demo does not contain any scam content or phishing links. It is a safe frontend mockup created for learning and presentation only.

## Files

- `index.html` — main web page markup
- `style.css` — visual layout and styling
- `assets/` — images used in the demo UI

## How to use

1. Open `index.html` in a browser.
2. Explore the interface and layout.

## Notes

- No real audio playback is implemented.
- The UI is for learning HTML/CSS and building a frontend clone.
